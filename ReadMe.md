# このソフトウェアについて #

ColorEditor20160621は私個人が学習目的で作成したソフトウェアです。

HSVまたはRGBで色を編集して表示します。

# 開発環境 #

* Windows XP SP3
* .NET Framework 4.0
* Windows Form Application
* C#
* Visual C# 2010 Express
